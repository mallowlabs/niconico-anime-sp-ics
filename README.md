niconico-anime-sp-ics
===================

Overview
----------------

.ics generater for niconico anime specials.

Authors
----------------

 * @mallowlabs

Requirement
----------------

 * Ruby 2.0.0

Install
----------------

Install dependencies:

    $ bundle install --path .bundle

Run:

    $ bundle exec foreman start

and access to http://localhost:5000/ics
